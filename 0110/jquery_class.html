  • addClass(); - 선택한 요소에 클래스 서식을 추가
  • removeClass(); - 선택한 요소에 클래서 서식을 제거
  • toggleClass(); - 선택한 요소에 클래스 서식을 추가/제거
  • hasClass(); - 선택한 요소에 해당 클래스 이름이 있는지 여부를 판단하여 true,fals 값으로 표시
  • class()l - 사용자가 선택한 요소에 서식을 적용
